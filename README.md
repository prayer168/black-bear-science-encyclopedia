# 黑熊老師自然科學繪本百科

> 用故事、圖解與實驗，陪孩子看懂自然世界。

一套為國小三年級到六年級設計、可銜接國中自然科學的「科普繪本百科」網站。
每篇文章都不是課本摘要，而是一篇有故事、有角色、有圖解、有實驗、有迷思破解的百科條目。

## 專案定位

- **對象**：國小三～六年級學生、可銜接國中的學生與教師
- **形式**：百科主題式 + 繪本故事式 + 科普圖解式
- **課程參考**：以 [115學年度國小國中自然課程架構](https://prayer168.github.io/science-course/)（康軒、翰林、南一）作為主題庫與課程銜接參考，但**不照課本單元順序**，改以百科主題重新整併。

## 年級設計方向

| 年級 | 定位 | 重點 |
|------|------|------|
| 三年級 | 自然觀察入門 | 觀察、分類、描述、生活經驗、簡單操作 |
| 四年級 | 生活現象解謎 | 發現自然現象背後的原因 |
| 五年級 | 科學偵探與變因探究 | 實驗設計、變因控制、證據解釋 |
| 六年級 | 系統思考與國中銜接 | 模型、系統、能量、交互作用與永續 |

## 九大主題分類

1. 植物與生命（plants-life）
2. 動物與昆蟲（animals-insects）
3. 空氣、水與天氣（air-water-weather）
4. 聲音、光與電（sound-light-electricity）
5. 力、運動與機械（force-motion-machines）
6. 物質、熱與變化（matter-heat-change）
7. 地表、岩石與自然資源（earth-resources）
8. 生態系與環境永續（ecology-sustainability）
9. 科學方法與探究實驗（inquiry-methods）

## 每篇文章固定結構（14 區塊）

1. 文章標題 → 2. 基本資料卡 → 3. 一句話吸引學生 → 4. 繪本式開場故事（300–500字）
→ 5. 角色對話（≥8輪）→ 6. 核心概念三層說明（小學生好懂版／高年級進階版／國中銜接版）
→ 7. 生活中的例子（3–5個）→ 8. 圖解設計（≥3張，含 AI 繪圖提示詞）
→ 9. 繪本分鏡（≥6頁）→ 10. 小實驗（含操縱／應變／控制變因）
→ 11. 迷思破解（≥5項）→ 12. 知識小測驗（5選擇＋3簡答＋2探究）
→ 13. 給老師的教學補充（含 Rubric）→ 14. 內容查核表

## 本機瀏覽與部署

- **本機**：直接雙擊 `index.html` 即可瀏覽（全站不使用外部 CDN，不需伺服器）。
- **GitHub Pages**：將本資料夾作為 repo 根目錄推上 GitHub，於 Settings → Pages 選擇 `main` 分支根目錄即可。

## 教師使用建議

- 依「年級」找符合班級程度的文章；依「主題分類」做跨年級課程設計。
- 每篇文章的「給老師的教學補充」可摺疊展開，含教學目標、引導問題、差異化建議、評量與 Rubric。
- 「小實驗區」皆為安全、低成本、可在教室或家中完成的活動，並標示三種變因。
- 「內容查核表」記錄該篇的科學查核狀態；標示「需要查證」者請勿直接授課使用。

## 品質原則

1. 不寫課本摘要、不照抄課程網站文字。
2. 科學正確優先；不確定的內容一律標示「需要查證」。
3. 語氣幽默、溫暖、清楚，適合國小三到六年級。
4. 清楚區分「觀察、推論、解釋」。
5. 圖像提示詞必須明確標示科學概念與「不能畫錯的地方」。
