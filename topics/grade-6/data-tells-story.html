資料會說話：圖表與統計初探

💡 一堆雜亂的數字，看起來頭昏眼花；但只要整理成表格、畫成圖表，它們就會「開口說話」，一眼看出誰多誰少、是升是降！這就是科學家最愛的祕密武器——資料與圖表。

📖 繪本開場：一堆看不懂的數字

自然課做完「一週天氣觀測」後，小狐同學的紀錄本上寫滿了數字：週一氣溫 22 度、雨量 5 毫米；週二 20 度、0 毫米；週三 18 度、12 毫米……密密麻麻。老師問：「這一週的天氣有什麼變化？」小狐盯著滿滿的數字，搔搔頭：「呃……好多數字，我看不太出來耶！」

黑熊老師走過來，笑著說：「數字這樣一條條列著，確實很難一眼看懂。但我教你一個魔法——把它們整理成表格，再畫成『圖表』，資料就會『開口說話』了！」他把小狐的氣溫資料畫成一條折線——「你看，這條線是不是一路往下？」小狐眼睛一亮：「真的耶！氣溫一天比一天低，一看就懂！」

接著黑熊老師又把雨量畫成一根根長條：「哪一天雨最多？」小狐立刻指出：「週三！它的長條最高！」黑熊老師點頭：「沒錯！同樣一堆數字，用對的圖表呈現，就能一眼看出誰多誰少、是升是降。這就是為什麼科學家做完觀察和實驗，都要把資料整理成圖表——因為圖表比一堆數字更會說話！」

水滴小隊長補充：「不過畫圖表也有學問喔——要選對種類（比較多少用長條圖、看變化趨勢用折線圖、看比例用圓餅圖），還要誠實、清楚，不能為了好看就亂改、故意誤導別人。今天我們就來學這個科學家的祕密武器：怎麼把資料整理成圖表、讓資料開口說話、又不被誤導！」

💬 角色對話時間

🦊 小狐同學

黑熊老師，為什麼要把資料整理成表格和圖表？直接看數字不行嗎？

🐻 黑熊老師

直接看一大堆數字，很難一眼看出重點！把資料整理成表格，可以讓數字排列整齊、方便對照；再畫成圖表，就能「一眼看出」誰多誰少、是升是降、佔多少比例。例如氣溫畫成折線，馬上看出升或降的趨勢；雨量畫成長條，馬上看出哪天最多。所以整理資料、畫圖表，是為了讓資料更容易看懂、比較和找出規律——這是做科學很重要的一步。

🦊 小狐同學

圖表有好幾種，我怎麼知道要用哪一種？

🐻 黑熊老師

要看你想表達什麼來選！①長條圖：適合比較不同項目的「多少」（例如各種水果誰最受歡迎、哪天雨最多），一根根比高矮很清楚。②折線圖：適合看「隨時間的變化趨勢」（例如一週氣溫、一個月的成長），線往上是增加、往下是減少。③圓餅圖：適合看「各部分佔整體的比例」（例如班上各種交通方式的佔比）。選對圖表，資料才會說得清楚；選錯了反而看不懂。

🪲 甲蟲博士

補一個很重要、卻常被忽略的觀念——圖表可以「誠實說話」，也可能「騙人」！ 例如：如果長條圖的縱軸不是從 0 開始，或刻度故意拉大縮小，本來差不多的兩根長條，看起來就會差很多，讓人誤以為差距很大。廣告、新聞有時就用這招誤導人。所以看圖表時，要看清楚：軸從哪開始？單位是什麼？資料來源可靠嗎？ 自己畫圖表時，更要誠實、清楚標示，不能為了好看或說服別人就故意扭曲。這是做科學和當個聰明公民都要有的素養！

🦊 小狐同學

那把資料畫成圖表以後，怎麼「讀」出結論呢？

🐻 黑熊老師

可以問自己幾個問題：①最大、最小的是哪個？（最高的長條、線的最高最低點）②有沒有明顯的變化趨勢？（一路上升、下降、還是起起伏伏）③各項目差多少、佔多少比例？④有沒有特別的、不尋常的點？把這些從圖表看到的「現象」整理出來，再連結到問題，就能下有根據的結論（例如「這週氣溫持續下降，週三雨量最多」）。記得結論要根據圖表上的證據，不能憑空亂猜。

🦊 小狐同學

學會看圖表，在生活中有什麼用呢？

🐻 黑熊老師

用處超大！新聞、報告、廣告裡到處都是圖表，看得懂才不會被誤導；做科展、報告時，用圖表呈現資料又清楚又有說服力；連看氣象、疫情、成績、零用錢紀錄，都能用圖表幫你看出趨勢、做決定。可以說，會整理資料、看懂圖表、又不被騙，是現代人非常重要的能力——因為我們活在一個「到處都是數據」的世界！

💧 水滴小隊長

我們來當資料魔法師！①把一份調查資料（如班上最喜歡的運動、一週氣溫）整理成表格。②選對圖表畫出來（比較用長條、趨勢用折線、比例用圓餅）。③從圖表找出最大最小、趨勢，寫出結論。④檢查自己的圖表有沒有誠實（軸從0、標清單位）。這個活動只處理資料、不涉及危險操作，很安全；但要誠實記錄與呈現，這是科學最重要的態度。

👾 迷思怪獸

哼，圖表隨便畫畫、好看就好啦！反正只要能證明我想講的，把長條拉高一點、軸動一下手腳，讓差距看起來大一點，有什麼關係～數字嘛，怎麼畫都行！

🐻 黑熊老師

迷思怪獸，這個想法很危險，一定要更正！圖表絕對不能為了「證明自己想講的」就動手腳、扭曲資料！如果縱軸不從 0 開始、刻度亂改，讓本來差不多的資料看起來差很多，那是在「用圖表騙人」——這違反科學最重要的「誠實」原則。科學是要誠實地呈現資料、讓證據說話，而不是先有結論、再去扭曲資料來配合。畫圖表要軸從 0（或清楚標示）、單位清楚、忠實呈現；如果資料不支持你原本的想法，那就修正想法，而不是竄改資料。誠實面對資料，才是真正的科學精神！

🦊 小狐同學

原來圖表要選對種類、還要誠實不騙人！我要把天氣資料畫成折線和長條，誠實地找出結論。

🧠 核心概念說明

🌱 小學生好懂版

做完觀察或實驗，會得到一堆資料（數字）。把它們整理成表格（排列整齊、方便對照），再畫成圖表，資料就會「一眼看懂」。要選對圖表：長條圖比較「誰多誰少」、折線圖看「隨時間的變化趨勢（升或降）」、圓餅圖看「各部分佔的比例」。看圖表時，找最大最小、變化趨勢、比例、特別的點，再根據這些證據下結論。最重要的是誠實：長條圖縱軸要從 0 開始、單位標清楚，不可以為了好看或說服別人就扭曲資料、故意誤導。看別人的圖表也要留意軸和刻度，才不會被騙。會整理資料、看懂圖表、不被誤導，是很重要的能力。

🌿 高年級進階版

科學探究蒐集到的原始資料需經整理（列表、分類、計數、可能求平均）再視覺化為圖表，以利比較、發現規律與呈現趨勢。選圖須配合資料性質與目的：長條圖比較類別的量、折線圖呈現連續（多為時間）變化、圓餅圖顯示部分對整體的比例。判讀時關注極值、趨勢、差距、比例與異常值，並將現象連結問題、以證據形成結論。須具備圖表識讀的批判力：檢視座標軸起點與刻度、單位、樣本大小與資料來源，避免截斷軸、變形比例等造成的誤導。製圖與詮釋皆應誠實、可重現，當資料不支持假設時應修正解釋而非竄改資料。此為科學論證與資訊素養的核心。

🌳 國中銜接版

資料處理流程：蒐集 → 整理（分類、次數分配、集中量數如平均數/中位數）→ 呈現（統計圖表）→ 分析 → 推論。圖表選擇依變數型態：類別資料用長條圖、圓餅圖；連續/時間序列用折線圖；分布用直方圖；關係用散布圖。判讀須辨識趨勢、相關、離群值，並區分相關不等於因果。常見誤導手法包括截斷 y 軸、非等距刻度、面積/立體變形、選擇性取樣，須加以辨識。科學論證要求以充分且忠實的證據支持主張，強調可重現性與同儕檢驗；資料造假違反研究倫理。國中數學「統計」與自然「科學探究與論證」會深入統計量、圖表與證據推理。

🏠 生活中的例子

  • 氣象圖：氣溫折線、雨量長條：一眼看出冷熱趨勢與哪天雨多。
  • 成績、身高的成長折線圖：看自己隨時間進步或長高的變化。
  • 班級調查（最愛的運動、交通方式）長條／圓餅圖：看誰最多、佔多少。
  • 零用錢收支表與圓餅圖：看錢花在哪、比例如何，幫助理財。
  • 新聞、疫情、選舉的統計圖：看懂才不會被誤導的圖表騙。
  • 科展、報告用圖表呈現實驗結果：清楚又有說服力。

🔬 水滴小隊長實驗室：資料魔法師——讓資料開口說話

適合年級六年級
材料一份調查或觀測資料（如班上最喜歡的運動人數、一週每天氣溫或雨量）、方格紙或畫圖工具、尺、彩色筆、計算機（求比例用）

操作步驟

  1. 整理成表格：把蒐集到的資料，整理成整齊的表格（項目、數量對照清楚）。
  2. 選對圖表：想想你要表達什麼——比較各項多少 → 長條圖；看隨時間變化 → 折線圖；看各部分佔比例 → 圓餅圖。
  3. 畫出圖表：畫圖時縱軸從 0 開始、標清楚單位和標題，資料忠實呈現，不誇大不縮小。
  4. 讀出結論：從圖表找出最大／最小、變化趨勢、比例、特別的點，寫成一兩句有根據的結論（例如「這週氣溫持續下降」）。
  5. 檢查誠實度：檢查自己的圖表——軸有沒有從 0？刻度平均嗎？單位清楚嗎？ 再互相看看別人的圖表，練習找出可能誤導的地方。

🔎 整理項目

把資料分類、計數，整理成表格。

📊 呈現方式

依目的選長條／折線／圓餅圖，忠實呈現。

✅ 誠實原則

軸從0、單位清楚、不扭曲，依證據下結論。

觀察紀錄表（範例）

項目／時間數量適合的圖表
各運動人數長條圖
一週氣溫折線圖
交通方式佔比圓餅圖

預期結果與科學解釋

把同一份資料整理成表格、畫成合適的圖表後，你會發現比一堆數字清楚太多：折線圖一眼看出上升或下降的趨勢、長條圖一眼看出誰多誰少、圓餅圖一眼看出各部分的比例。你可以根據圖表上的證據（最大、最小、趨勢）寫出有根據的結論。同時你會體會到：選對圖表資料才說得清楚；而如果把縱軸截斷或刻度動手腳，同樣的資料看起來就會被扭曲、誤導人。這說明：整理與呈現資料要選對方式、並且誠實，才能讓資料正確地「說話」——這正是科學探究「分析資料、以證據下結論」的方法。

⚠️ 提醒（科學誠信）：這個活動沒有危險操作，但有一個最重要的原則——誠實！記錄與呈現資料時，不可以捏造、竄改或故意用圖表誤導。如果資料的結果和你原本的想法不一樣，要修正想法、如實呈現，而不是改資料來配合。誠實面對證據，是科學（也是做人）最重要的態度。看別人的圖表也要有判斷力，不被誤導。

👾 迷思怪獸來搗蛋（迷思破解）

👾 迷思說法：圖表隨便畫、好看就好，反正能證明我想講的就行。

為什麼容易誤解：覺得圖表只是裝飾。

✅ 正確說法：圖表不能為了證明自己想講的就扭曲資料！科學要誠實呈現資料、讓證據說話。如果先有結論再去改資料配合，就是造假、騙人，違反科學最重要的誠實原則。畫圖表要忠實、清楚；資料不支持想法時，要修正想法而不是改資料。

可以怎麼驗證：科學重視可重現與同儕檢驗，造假會被揭穿。

👾 迷思說法：什麼資料都用同一種圖表就好，例如全部用長條圖。

為什麼容易誤解：覺得圖表都差不多。

✅ 正確說法：要依目的選對圖表！比較多少用長條圖、看隨時間的變化趨勢用折線圖、看各部分佔整體的比例用圓餅圖。用錯圖表（例如用圓餅圖看趨勢），資料就說不清楚、甚至看不懂。選對圖表，資料才能好好說話。

可以怎麼驗證：同一份時間資料用折線清楚、用圓餅就看不出趨勢。

👾 迷思說法：只要有圖表，內容就一定是對的、可以相信。

為什麼容易誤解：圖表看起來很專業。

✅ 正確說法：有圖表不代表一定正確！圖表可能被動手腳誤導（縱軸不從 0、刻度不平均、資料來源不可靠、只選對自己有利的資料）。看圖表要留意：軸從哪開始？單位是什麼？資料哪來的？有沒有以偏概全？ 才不會被騙。

可以怎麼驗證：檢查廣告或新聞圖表的軸與來源，常有誤導。

👾 迷思說法：長條圖的縱軸從哪裡開始都沒差。

為什麼容易誤解：覺得只是起點不同。

✅ 正確說法：差很多！長條圖的縱軸應該從 0 開始。如果從中間某個數字開始（截斷軸），本來差不多的兩根長條，看起來就會差距被放大，讓人誤以為差很多。這是常見的誤導手法，所以畫圖要從 0、看圖要注意軸。

可以怎麼驗證：同資料軸從0與截斷軸並排，差距觀感完全不同。

👾 迷思說法：從圖表看到「兩件事一起變」，就代表一個造成另一個。

為什麼容易誤解：兩條線一起升降，好像有關。

✅ 正確說法：「一起變化」不一定是「誰造成誰」！ 兩件事同時上升或下降，可能只是巧合，或另有共同的原因。要下「因果」結論，需要更多證據和實驗，不能光看圖表上一起變就斷定。（這叫「相關不等於因果」，國中會深入。）

可以怎麼驗證：找一起變化卻無因果的例子（如冰淇淋銷量與溺水）。

選擇題

1. 為什麼要把資料整理成表格、畫成圖表？

看答案與解析

答案：B。圖表比一堆數字更容易看出重點與趨勢。

(A) 圓餅圖 (B) 折線圖 (C) 長條圖比較 (D) 表格就好

看答案與解析

答案：B。折線圖適合呈現隨時間的變化趨勢。

(A) 折線圖 (B) 圓餅圖 (C) 溫度計 (D) 地圖

答案：B。圓餅圖適合呈現各部分佔整體的比例。

4. 畫長條圖時，縱軸應該從哪裡開始才誠實？

答案：A。縱軸從0才不會放大差距、誤導人。

看答案與解析

答案：B。誠實呈現、讓證據說話是科學精神。

簡答題

1. 為什麼「一堆數字」要整理成表格和圖表？這樣做有什麼好處？

參考答案

因為一大堆零散的數字很難一眼看出重點和變化。整理成表格，可以讓數字排列整齊、方便對照；再畫成圖表，就能一眼看出誰多誰少（長條圖）、隨時間是升是降的趨勢（折線圖）、各部分佔多少比例（圓餅圖）。好處是讓資料更容易看懂、比較和找出規律，也更能清楚地呈現給別人看，方便根據資料下結論。這是科學研究整理與呈現資料的重要方法。

2. 長條圖、折線圖、圓餅圖各適合用來表達什麼？請各舉一個例子。

參考答案

長條圖：適合「比較不同項目的多少」，例如比較班上各種最喜歡的運動各有幾人、或哪一天雨量最多。折線圖：適合看「隨時間的變化趨勢」，例如一週每天的氣溫變化、或自己一年來身高的成長。圓餅圖：適合看「各部分佔整體的比例」，例如班上同學各種上學交通方式（走路、坐車、騎車）各佔多少比例。要依照想表達的目的來選對圖表。

3. 為什麼看圖表（或畫圖表）時，要特別注意「縱軸從哪裡開始」？

參考答案

因為縱軸的起點會影響我們對資料差距的感覺。長條圖的縱軸應該從 0 開始，這樣長條的高矮才能忠實反映真正的數量差距。如果縱軸不從 0、而是從中間某個數字開始（截斷軸），本來差不多的兩根長條，看起來就會差距被放大很多，讓人誤以為差很大。這是常見的誤導手法。所以畫圖表要從 0、忠實呈現；看別人的圖表也要注意縱軸起點和刻度，才不會被誤導。

探究思考題

1. 小狐做了「植物在不同顏色光下生長」的實驗，量了每株的高度。他想在報告裡「一眼呈現哪種顏色長最高」，也想呈現「每株一個月來每週的高度變化」。這兩個目的，各應該用什麼圖表？為什麼？

思考方向

「一眼呈現哪種顏色長最高」（比較不同顏色光下最後的高度）→ 用長條圖：把每種顏色的高度畫成一根根長條，比高矮就能一眼看出哪種顏色長最高，適合「比較不同項目的多少」。「每株一個月來每週的高度變化」（隨時間的成長趨勢）→ 用折線圖：把每週的高度連成一條線，可以清楚看出植物一週一週長高的趨勢（線往上升代表持續長高），適合呈現「隨時間的變化」。所以要依照目的選圖表：比較用長條、看趨勢用折線。（能正確配對並說明理由即可。）

2. 某飲料廣告畫了一張長條圖，說自己的產品「銷量遙遙領先」，兩根長條差很多。小狐仔細一看，發現縱軸是「從 90 開始」，其實兩家銷量是 95 和 100。請你分析這張圖哪裡有問題？正確應該怎麼畫？

思考方向

問題在於這張長條圖的縱軸沒有從 0 開始，而是從 90 開始（截斷軸）。實際上兩家的銷量是 95 和 100，只差 5、差距很小；但因為縱軸從 90 開始，長條只畫「超過 90 的部分」（一根 5、一根 10），看起來一根是另一根的兩倍高，把小小的差距放大了，讓人誤以為「遙遙領先」。這是用圖表誤導人的手法。正確的畫法應該是縱軸從 0 開始，這樣 95 和 100 兩根長條會幾乎一樣高，忠實呈現「其實差不多」的真相。看廣告或新聞的圖表時，要特別注意縱軸的起點，才不會被騙。（能指出截斷軸放大差距、應從0畫即可。）
