# 繪本分鏡模板

> 每篇文章至少 6 頁分鏡。目標：讓插畫家或 AI 繪圖能直接照做，且不畫錯科學概念。

## 繪本基本資料

- 繪本標題：
- 對應文章：
- 適合年級：
- 頁數：（≥6）
- 固定登場角色：（🐻 黑熊老師／🦊 小狐同學／🪲 甲蟲博士／💧 水滴小隊長／👾 迷思怪獸）
- 整體畫風：溫暖手繪水彩風、明亮色調、角色表情豐富、適合國小生

## 分鏡頁模板（每頁複製一份）

### 第 N 頁

| 欄位 | 內容 |
|------|------|
| 頁碼 | N |
| 畫面描述 | （場景、構圖、角色位置、動作、表情） |
| 旁白 | （1–3 句，說故事的口吻） |
| 角色台詞 | （角色名：台詞） |
| 科學概念 | （這一頁要讓讀者理解什麼） |
| 圖像提示詞 | （英文或中文 AI 繪圖提示詞，需包含科學元素） |
| 不能畫錯的地方 | （明確列出：例「昆蟲只能畫六隻腳」「箭頭方向是被吃者指向捕食者」） |

## 分鏡節奏建議

1. 第 1 頁：情境與角色登場（引起好奇）
2. 第 2 頁：問題出現（小狐同學的疑問或迷思怪獸的錯誤說法）
3. 第 3–4 頁：觀察與探究（水滴小隊長帶領實驗或觀察）
4. 第 5 頁：概念揭曉（黑熊老師解釋＋甲蟲博士冷知識）
5. 第 6 頁：總結與延伸（回到生活、留下一個新問題）
