# 圖像提示詞模板

> 每張圖解與分鏡插圖都用本模板撰寫，確保「科學正確」優先於「畫面好看」。

## 提示詞結構（七要素）

```
【圖名】
【圖像類型】概念總覽圖 / 過程機制圖 / 實驗應用圖 / 繪本分鏡插圖
【畫風】溫暖手繪水彩繪本風，明亮色調，白色背景，適合國小科普讀物
【主體】（畫面中心是什麼）
【必要科學元素】（缺一張圖就失敗的元素，逐項列出）
【必須標示的科學概念】（圖中要有的文字標籤或箭頭，含方向）
【構圖】（左右/上下/環狀/剖面/放大鏡特寫等）
【禁止畫錯】（負面清單，逐項列出）
```

## 範例

```
【圖名】光合作用的原料與產物總覽圖
【圖像類型】概念總覽圖
【畫風】溫暖手繪水彩繪本風，明亮色調，白色背景
【主體】一株盆栽植物，葉片放大特寫
【必要科學元素】太陽、箭頭指向葉片的「陽光」、由根吸入的「水」、
由氣孔進入的「二氧化碳」、由葉片釋出的「氧氣」、葉內生成的「養分（葡萄糖）」
【必須標示的科學概念】原料（水、二氧化碳、光）在箭頭指入側；
產物（氧氣、養分）在箭頭指出側；標示「光合作用發生在葉子的葉綠體」
【構圖】植物置中，左側輸入箭頭、右側輸出箭頭
【禁止畫錯】不可把「土壤」畫成養分來源；不可讓植物「吃土」；
箭頭方向不可顛倒；不可畫成夜晚場景
```

## 通用禁止清單（所有圖共用）

- 昆蟲一律六隻腳、三個體段（頭、胸、腹），腳都長在胸部。
- 蜘蛛（八隻腳）不可標示為昆蟲。
- 食物鏈箭頭：被吃的生物 → 吃它的生物（代表能量流向），不可反向。
- 熱傳導、對流、輻射三種方式不可在同一圖中混淆標示。
- 太陽不可畫成「有笑臉就代表白天溫暖」之類與概念無關卻誤導的擬人（角色擬人可以，科學物件標示要準確）。
- 圖中文字標籤使用繁體中文（台灣用語）。
